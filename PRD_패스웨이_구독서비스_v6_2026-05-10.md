## 6. 거래량 3대 지표 (v6 NEW ★★★)

### 6.1 CMF (Chaikin Money Flow, 20일)

```
MFM = ((close - low) - (high - close)) / (high - low)   # Money Flow Multiplier
MFV = MFM × volume                                       # Money Flow Volume
CMF20 = sum(MFV, 20D) / sum(volume, 20D)                # -1 ~ +1
```
**기준선**:
- ≥ +0.25 = 강한 매집 (책에서 말하는 황금 구간)
- ≥ 0 = 매집 진행
- ≤ -0.25 = 강한 분산

**활용**: 그루 시그널 결합 필터의 핵심 지표 (옵션 B 채택)

### 6.2 A/D Line (Accumulation/Distribution Line, 누적)

```
AD = cumulative_sum(MFV)
```
**해석**: 절대값은 의미 없음. **방향성·다이버전스**로 해석.

**S3 강세 다이버전스 시그널 (v6 신규)**:
```
가격이 20D 신저가 AND A/D는 20일 전 대비 상승
→ "바닥 다지기 끝, 매집 진행 중"
```
- 단독 백테스트: 음수비 41% (안정성 1위), 평균 +4.79%
- 그루와 동일자 결합 X (구조적으로 신저가 vs 신고가 동시 불가)
- "최근 20일 내 발생 여부"를 정황 뱃지(🔄)로 그루 종목 옆에 표시

### 6.3 MFI (Money Flow Index, 14일)

```
TP = (high + low + close) / 3                        # Typical Price
MF = TP × volume                                      # Money Flow
PMF = sum(MF where TP > prev_TP, 14D)                # Positive MF
NMF = sum(MF where TP < prev_TP, 14D)                # Negative MF
MFI = 100 - 100 / (1 + PMF/NMF)                      # 0 ~ 100
```
**기준선**: 80 과매수 / 50 중립 / 20 과매도

**활용**: 차트 보조 표시 전용 (그루 결합 효과 미미 → 결합 필터에서 제외)
